Umigon - free sentiment analysis for Tweets / Twitter

Umigon

Sentiment analysis for tweets, and more

Or:





Application created by Clement Levallois. Can process millions of tweets in minutes. Download the (not too technical) paper describing how Umigon works.

Code available in open source on Github. Interested in integrating this app in your products (via API etc.) or research projects? Contact me.

If you use this application, please reference it in your work:

Levallois, Clement. "Umigon: sentiment analysis on Tweets based on terms lists and heuristics". Proceedings of the 7th International Workshop on Semantic Evaluation (SemEval 2013), June 2013, Atlanta, Georgia.
